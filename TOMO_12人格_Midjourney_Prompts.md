## 使用说明

1. **直接复制 ``` 代码块内文字** 粘贴到 Midjourney
2. 建议先用 `--v 6.1` 出图，效果不满意可切 `--v 6.0` 或 `--niji 6`
3. 如需更暗背景，可在 prompt 末尾加 `--style raw`
4. 如需去掉某些不想要的元素，用 `--no red, orange, yellow, brown, warm colors` 强制排除暖色
5. 出图后用 `U1-U4` 放大喜欢的版本，再用 `Vary (Subtle)` 微调

**推荐追加参数（所有 prompt 通用）：**
```
--no red, orange, yellow, brown, warm tones, fire
```
这会确保所有出图都不会出现风水禁忌色。
